I was initially tasked with creating a basic portfolio using WordPress. Instead of keeping it generic, I decided to turn it into a fun and creative project — a character portfolio for Geralt of Rivia from The Witcher 3.
The design is inspired by Geralt's aesthetics in The Witcher 3, and the content structure was planned by researching official wiki pages and in-game references. This project showcases my ability to build themed, content-driven websites using WordPress while maintaining visual consistency and usability.

Tools Used : Wordpress and Superb Addon Themes
